Quick Guide to Using the TNRS

The TNRS allows you to validate and correct a list of plant names against an authoritative database of published scientific names and authorities.

To view more detailed instructions, click here: Instructions

You may enter names on the fly by typing or pasting up to 5000 names in the Enter List tab.

Alternatively,you can upload a text file with an unlimited number of names in the Upload and Submit List tab. Your file MUST be plain text, with one name per line, and must have a .txt filename extension; word processor, or spreadsheet files are NOT supported. You will received an email notification when the list is done being processed. You can retrieve your results at that time in the Retrieve Results tab.
